Functions in OSWorkflow are where you can perform the "meat" of your workflow-based application. They can be executed before and after (pre- and post-) transitions from one state to another in the finite state machine. OSWorkflow supports the following forms of functions:

OSWorkflow Java-based Function

  • Java classes loaded by a ClassLoader
  • Java classes retrieved via JNDI
  • Remote EJBs
  • Local EJBs

OSWorkflow BeanShell Functions

OSWorkflow BSF Functions (perlscript, vbscript, javascript)

OSWorkflow Utility Functions

OSWorkflow also includes a small set of OSWorkflow Utility Functions that will start you off in the right direction. Some of these utility functions are extremely valuable in creating dynamic workflow definitions.
