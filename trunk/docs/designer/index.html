About

The OSWorkflow Designer is a Swing based GUI application to edit and manage workflows. It is currently in an alpha stage, but is already fairly functional

Try It!

A webstart version of the designer is currently available. To launch it, click here.

Workspace

Before you get to the fun of editing workflows, you will need to create a workspace. A workspace is simply a convenient way of grouping together multiple workflows. To create a workflow, select File -> New -> New workspace. This will create a new unsaved workspaces, that you can import workflows into.

Before importing though, you need to save the workspace. Select File -> Save workspace. You will be prompted for a filename. Note that the designer will save all the project files alongside the workspace file, so it is recommended that you create a new empty directory and save the workspace into it.

Now that your workspace is saved, you can import or create workflows. For a large collection of example workflows to import, look in OSWorkflow's src/test/samples directory. You can download these samples from here

As noted earlier, the designer is an alpha release, so new features and fixes are happening daily. If you do find any bugs or have any contributions you'd like to make, then please do not hesitate to file an issue for it. Enjoy!
