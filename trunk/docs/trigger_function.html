Trigger function(s) are defined in workflow definition xml file. Parent-child relationships are specified in the http://www.opensymphony.com/osworkflow/workflow_2_5.dtd

Description of the trigger function can be found in OSWorkflow Workflow Concepts, and a usage suggestion in OSWorkflow Utility Functions page. Currently there is no trigger function example included in osworkflow-2.5.0-example workflow definition example.xml file, you have to add it yourself if you want to test it with Quartz scheduler code.
